# AI PDF Question Answering using Endee

## Overview
This project demonstrates a Retrieval-Augmented Generation (RAG) system using Endee as a vector database.

## Features
- Load PDF documents
- Extract text
- Prepare for semantic search
- Basic AI pipeline

## Tech Stack
- Python
- LangChain
- Endee (Vector Database)

## How it Works
1. Load PDF file
2. Extract text from document
3. Store embeddings in Endee
4. Retrieve relevant information

## Setup
pip install -r requirements.txt  
python app.py  

## Endee Usage
Endee is used as a vector database to store and retrieve embeddings efficiently.
## Conclusion
This project demonstrates a basic implementation of Retrieval-Augmented Generation (RAG) using Endee as a vector database. It enables semantic search over document data.
